Venture Jobs Foundation Homepage Redesign

  • next
  • prev

The focus of this project was to redesign a local homepage. I wanted to test a responsive framework with a corporate look and feel. I also created a new style guide that followed the original branding.

Responsive Design

I used the Foundation framework to help layout my responsive design. The homepage is viewable on mobile phones and tablets.
Click Here.



Process

It was fun learning the specific media queries associated with mobile phones and tablets. With different views set for each device I had to design a way to be able to seamlessly change between three options.

With a focus on touch devices, the navigation needed to be touch friendly. On mobile or tablet viewports the navigation is set to change to a touch friendly system.
